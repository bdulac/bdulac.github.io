In a few words, what is Java ?

Java is the trademark of a software platform consisting in three sets of elements:

  1. A programming language
  2. A virtual machine (JVM)
  3. Runtime environments (JRE and others)

All three are defined in specifications JCP + specifications = answer(JSR)

Runtime environments are declined in three versions (Java ME, Java SE and Java EE)
