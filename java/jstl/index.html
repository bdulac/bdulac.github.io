"fn" (functions)

There is a problem in testing the content HTTP request URL: the HttpServletRequest method returns a StringBuffer. But the fn:contains(.., ..) functions accepts only strings. This works but is not static-type conform.

The solution is to set the request URL in a variable to transform it into a string. Then the function can be used properly 

<c:set var="requestUrl" value="${pageContext.request.requestURL}" />
		<c:if test"${fn:contains(requestUrl, '...')}">
		</c:if>
