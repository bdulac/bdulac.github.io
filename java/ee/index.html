Java Enterprise Edition (Java EE) formerly named J2EE is a software development platform for enterprise software components. It is defined as a server-side specifications gathering other component Java specifications. Since 1999, six editions of the platform have been specified:

  • J2EE 1.2
  • J2EE 1.3
  • J2EE 1.4
  • Java EE 5
  • Java EE 6
  • Java EE 7

Here are some personal documentations about specifications part of the Java EE platform:

  • Java API for RESTful Services (JAX-RS)
  • JSP Standard Tag Library (JSTL)
